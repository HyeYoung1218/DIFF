# DIFF (SIGIR'25)
This is the official repository for our paper "**DIFF: Dual Side-Information Filtering and Fusion for Sequential Recommendation**" in SIGIR'25.
You can find our paper on [[arXiv](https://arxiv.org/abs/2505.13974)].
